# Hospital Management System

A web-based Hospital Management System built using:

- Java (JSP & Servlets)
- JDBC
- MySQL
- Bootstrap
- Apache Tomcat
- Eclipse IDE

## Features

Admin Panel
- Manage doctors
- Manage specialists
- View appointments

Doctor Panel
- View patient appointments
- Approve / reject appointments
- Edit profile

User Panel
- Register / Login
- Book appointments
- View appointment history
- Change password

## Database
MySQL

## Server
Apache Tomcat 9
